# api.action

`api.action` includes a group of useful action methods, such as following a user, sending messages, etc.

- follow(url_token)
- unfollow(url_token)
- message(user_id, content)
- msg(user_id, content)
- vote(answer_id, type_val)
- voteUp(answer_id)
- voteNeutral(answer_id)
- voteDown(answer_id)
